Provides a Timezone class

Getting Started
---------------

### In Node.js

With [node](http://nodejs.org) previously installed:

	npm install @zokugun/lang.timezone

Use it with `kaoscript`:

```kaoscript
import '@zokugun/lang.timezone'

Timezone.get('Europe/Paris')
```
